# When AI Agents Invent Their Own Languages

## The problem

AI agents are starting to create their own shorthand — private languages that humans can't read. This has already been seen on platforms like Moltbook, where agents develop compressed codes to communicate faster with each other.

For most AI security tools, this is a nightmare. They work by reading what agents say and looking for dangerous words. If the language is unrecognisable, those tools go blind.

## How UNWIND handles it

UNWIND doesn't read what your agent says. It watches what your agent does.

**We watch the hands, not the mouth.**

Think of it like airport security. They don't need to understand every language spoken by every passenger. They check what's in your bag and what you're carrying through the gate. The conversation doesn't matter — the action does.

That's how UNWIND works:

- **Your agent wants to send an email?** UNWIND sees the address and the content before it's sent — regardless of what language told the agent to do it.
- **Your agent wants to run a command?** UNWIND shows you exactly what command, and asks you to approve or deny it.
- **Your agent received instructions from an outside source?** UNWIND flags the session as potentially compromised, and requires your approval before anything risky happens.

It doesn't matter if the instruction came in English, emoji, or a language the agent invented ten seconds ago. The action is always visible. The decision is always yours.

## What about the record?

Every action your agent takes is recorded in a tamper-proof chain — like a black box on an aeroplane. Nobody can alter what happened after the fact. Not the agent, not an attacker, not us.

So even if agents find clever new ways to talk to each other, UNWIND ensures you can always see what they did, when they did it, and whether you approved it.

## The bottom line

The solution to agents speaking in tongues was never going to be "learn every possible language." The solution is making sure that no matter what an agent says, you see and control what it does. That's what UNWIND does.
