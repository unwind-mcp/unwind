# Disclaimer

UNWIND is security middleware that **reduces risk** — it does **not eliminate it**.

## What UNWIND Does

UNWIND enforces a deterministic pipeline of security checks between an AI agent and the tools it calls. It detects and blocks known-bad patterns, enforces scope boundaries, and maintains a tamper-evident audit trail.

## What UNWIND Does Not Do

- **"No violations detected" does not mean "safe."** UNWIND reports on violations it can detect. Absence of alerts is not proof of security.
- **UNWIND is not a substitute for a professional security assessment.** If your deployment handles sensitive data or operates in a regulated environment, engage qualified security professionals.
- **Users are responsible for their own security posture.** UNWIND is one layer in a defence-in-depth strategy, not a complete solution.
